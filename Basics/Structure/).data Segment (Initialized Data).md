

ဒီအပိုင်းမှာ Program ရေးတုန်းက တန်ဖိုးတစ်ခုခု ကြိုသတ်မှတ်ပေးထားတဲ့ (initialized ဖြစ်ထားတဲ့) **Global Variables** တွေနဲ့ **Static Variables** တွေကို သိမ်းပါတယ်
Read-Write section ဖြစ်တယ် Executable တော့ မဖြစ်
ASLR On / No-PIE မှာ Address fixed, PIE မှာ base address ပေါ်မူတည်
Size က compile time မှာ သိရပြီး runtime မှာ ပြောင်းလဲလို့ရတယ်

Global Variables: Function တွေရဲ့ အပြင်မှာ ကြေညာထားတဲ့ variable တွေ
Static Variables: `static` keyword သုံးပြီး ကြေညာထားတဲ့ variable တွေ

| Feature           | Description                                               |
| ----------------- | --------------------------------------------------------- |
| **Content**       | Initialized global & static variables (non-zero)          |
| **Initial value** | Explicit values from code (100, 3.14, 'A', "hello", etc.) |
| **File size**     | Binary file ထဲမှာ actual space ယူတယ်                      |
| **Memory**        | Program load ချိန်မှာ value တွေ copy လုပ်                 |
| **Read/Write**    | ✅ Read and Write allowed                                  |
| **Address**       | Usually between TEXT and BSS                              |

```
High address
+-----------------+
|      Stack      |  (local variables, return addresses)
|        ↓        |
|        ↑        |
|      Heap       |  (malloc)
+-----------------+
|      .bss       |  ← uninitialized/zero-initialized (read-write)
+-----------------+
|      .data      |  ← initialized non-zero (read-write)
+-----------------+
|      .rodata    |  ← read-only data (strings, constants)
+-----------------+
|      .text      |  ← code (read-only)
+-----------------+
Low address
```



##### Initialized Global Variables

Function တွေအားလုံးရဲ့ အပြင်ဘက်မှာ ကြေညာထားပြီး၊ သုည (၀) မဟုတ်တဲ့ တန်ဖိုးတစ်ခုခု ကြိုပေးထားတဲ့ variable တွေ ဖြစ်ပါတယ်

```c
int global_counter = 10;  // ✅ .data section သို့ ရောက်သည် (Global ဖြစ်ပြီး initialized လုပ်ထား၍)
char status_flag = 'A';   // ✅ .data section သို့ ရောက်သည်

void main() {
    // ...
}
```

##### Initialized Static Variables (Local ကော Global ကော)

သူတို့ရဲ့ Scope က function ထဲမှာပဲ ဖြစ်ဖြစ်၊ တစ်ဖိုင်လုံးမှာပဲ ဖြစ်ဖြစ် `static` လို့ သတ်မှတ်ထားပြီး တန်ဖိုးတစ်ခုခု ပေးထားရင် `.data` ထဲ ရောက်ပါတယ်

```c
void count_func() {
    static int call_count = 1; // ✅ .data section သို့ ရောက်သည် (Static ဖြစ်ပြီး initialized လုပ်ထား၍)
    call_count++;
}
```

##### Initialized Global / Static Arrays နှင့် Struct များ

တစ်လုံးချင်းစီတင်မကဘဲ တန်ဖိုးအသေ ထည့်ပေးထားတဲ့ array တွေနဲ့ struct တွေကလည်း writable ဖြစ်တဲ့အတွက် `.data` ထဲမှာ နေရာယူပါတယ်

```c
int my_scores[3] = {85, 90, 95}; // ✅ .data section သို့ ရောက်သည်
```

```c
// 11. STRING LITERALS as arrays (global)
char global_literal_arr[] = "hello";      // DATA (copy of "hello")
char *global_literal_ptr = "hello";       // global_literal_ptr → DATA

// 12. CONST arrays (global) - actually in TEXT, not DATA!
const int const_arr[3] = {1, 2, 3};       // ⚠️ TEXT/rodata, not DATA!
```

##### not in data section

|Data Type|Storage|Why not DATA?|
|---|---|---|
|`int g;` (no init)|**BSS**|Uninitialized (zero)|
|`static int x;` (no init)|**BSS**|Uninitialized static|
|`int x = 0;` (global)|**BSS** (optimization)|Zero-initialized goes to BSS|
|`const int x = 5;`|**TEXT/rodata**|Read-only|
|`int x = 5;` (local)|**STACK**|Local variable|
|`char *p = "hi";` (local)|**STACK** (p) + **TEXT** ("hi")|Local pointer|
|`malloc()`|**HEAP**|Dynamic|
|`static int x = 0;`|**BSS**|Zero-init static|

##### DATA vs BSS vs TEXT Comparison

|Aspect|DATA|BSS|TEXT/rodata|
|---|---|---|---|
|**Initial value**|Explicit (non-zero)|0 (implicit)|Explicit (read-only)|
|**Binary size**|Takes space|No space (only size)|Takes space|
|**Example**|`int g = 5;`|`int g;`|`const int g = 5;`|
|**Can modify?**|✅ Yes|✅ Yes|❌ No|
|**Zero-init optimization**|No|Yes (int x=0 → BSS)|N/A|

```c
#include <stdio.h>
#include <stdlib.h>

// ===== DATA VARIABLES (with explicit non-zero values) =====
int data_int = 100;                    // 100
int data_int_negative = -50;           // -50
unsigned int data_uint = 4294967295U;  // 4294967295
float data_float = 3.14159f;           // 3.14159
double data_double = 2.718281828;      // 2.718281828
char data_char = 'Z';                  // 'Z' (ASCII 90)
char data_str[10] = "DATA";            // 'D','A','T','A','\0',0,0,0,0,0
int data_arr[5] = {10, 20, 30, 40, 50}; // 10,20,30,40,50

// ===== DATA STRUCTURES =====
typedef struct {
    int id;
    char name[20];
    float score;
} Student;

Student data_student = {101, "Alice", 95.5};  // DATA

// ===== DATA POINTERS =====
int *data_ptr = &data_int;             // Stores address of data_int
char *data_str_ptr = "string literal"; // Pointer in DATA, string in TEXT

// ===== STATIC DATA (file scope) =====
static int static_data = 999;          // DATA (file scope)
static char static_data_arr[] = "static data"; // DATA

// ===== FUNCTION POINTER =====
void my_function() { }
void (*func_ptr)(void) = my_function;  // DATA

// ===== ENUM =====
enum Status { ACTIVE, INACTIVE, PENDING };
enum Status data_status = ACTIVE;      // DATA (value 0)

void function() {
    // ===== STATIC LOCAL (initialized) =====
    static int counter = 0;             // DATA (not stack!)
    static char buffer[20] = "local";   // DATA
    static float pi = 3.14;             // DATA
    
    counter++;
    printf("Counter: %d (stored in DATA at %p)\n", counter, &counter);
    
    // ===== LOCAL VARIABLES (STACK, not DATA) =====
    int local = 50;                     // STACK
    char local_arr[] = "stack";         // STACK
}

int main() {
    printf("========== DATA SEGMENT VARIABLES ==========\n");
    printf("data_int = %d at %p\n", data_int, &data_int);
    printf("data_float = %f at %p\n", data_float, &data_float);
    printf("data_char = '%c' at %p\n", data_char, &data_char);
    printf("data_str = '%s' at %p\n", data_str, data_str);
    printf("data_arr[0] = %d at %p\n", data_arr[0], &data_arr[0]);
    printf("data_student.id = %d at %p\n", data_student.id, &data_student.id);
    printf("data_ptr = %p (points to %p)\n", data_ptr, &data_int);
    printf("static_data = %d at %p\n", static_data, &static_data);
    
    printf("\n========== MODIFYING DATA SEGMENT ==========\n");
    printf("Before: data_int = %d\n", data_int);
    data_int = 999;  // ✅ Allowed (DATA is writable)
    printf("After: data_int = %d\n", data_int);
    
    printf("\n========== STATIC LOCAL PERSISTENCE ==========\n");
    function();  // counter: 1
    function();  // counter: 2
    function();  // counter: 3
    
    // ⚠️ Cannot modify TEXT segment
    // const int *p = &some_const;
    // *p = 10;  // 💥 Segmentation fault
    
    return 0;
}
```

```c
// Case 4: Local pointer to literal (pointer in STACK)

    char *ptr = "hello";   // Pointer on stack, points to TEXT
```

#### Overwrite 
`.data` က **writeable** ဖြစ်လို့ exploitation မှာ target တစ်ခုဖြစ်တယ်။
- Global flag variable `is_admin = 0` ကို overflow ကနေ 1 ပြောင်းပြီး privilege bypass လုပ်တာ။
- Function pointer တစ်ခု `.data` ထဲရှိနေရင် shellcode ဆီ ညွှန်ပြောင်းလို့ရတယ်။
    
#### GOT 
- GOT က `.got` သို့ `.got.plt` section ထဲမှာရှိတယ်။
- Ret2libc, GOT overwrite attack တွေမှာ `.data` နားကို ရောက်တတ်တယ်။
- Format string vulnerability နဲ့ GOT entry ကို system address ပြောင်းနိုင်တယ်။
    
####  Information Disclosure
- `.data` ထဲမှာ interesting data (passwords, keys, function pointers) ရှိနိုင်တယ်။
- Leak လုပ်နိုင်ရင် ASLR bypass အတွက် အသုံးဝင်တယ်။
    
#### Relocation & Dynamic Linking
- `.data` ထဲက pointers တွေက runtime မှာ fix up ခံရတယ်။
- Partial overwrite (e.g., last 2 bytes) လုပ်ပြီး ROP chain ဆီပြောင်းနိုင်တယ်။



```c
#include <stdio.h>
#include <string.h>

char target[64] = "normal";  // .data ထဲမှာ

int main(int argc, char **argv) {
    char buffer[64];
    strcpy(buffer, argv[1]);   // buffer overflow ရှိတယ်
    printf("Target is %s\n", target);
```
