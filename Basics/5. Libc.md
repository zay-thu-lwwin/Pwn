#### Dynamic Linker/Loader

	program တစ်ခုစဖွင့်ဖွင့်ချင်းမှာ လိုအပ်တဲ့ shared libraries တွေကို load လုပ်ပေးတယ်
	memory ထဲမှာ program ကို ပြင်ဆင်ထည့်သွင်းပေးတယ်
	


အကြမ်းဖျင်းအားဖြင့်

```
Step 1: Kernel → ld-2.23.so (dynamic linker ကို load)
Step 2: ld-2.23.so → libc-2.23.so ကို load
Step 3: ld-2.23.so → သင့် program ကို run
Step 4: သင့် program → libc-2.23.so ထဲက functions တွေသုံး
```

## How it works together ??

	./program ဆိုပီး စ run တဲ့အခါမှာ

#### Kernel က စတင်ခြင်း
	 Kernel က execve() system call ကို အသုံးပြုပြီး program ကို load လုပ်တယ်
	 ELF (Executable and Linkable Format) header ကို ဖတ်တယ်
	 INTERP section ထဲမှာ dynamic linker path ကို ရှာတယ် 
	(အများအားဖြင့် /lib/ld-linux.so.2)
#### Dynamic Linker ld.soစတင်ခြင်း
	 Kernel က ld-2.23.so (သို့) ld.so.2 ကို memory ထဲ load လုပ်တယ်
	 Dynamic linker က control ကို ယူလိုက်တယ်
	 Program ရဲ့ PT_LOAD segments တွေကို memory ထဲ map လုပ်တယ်

#### Dependency ရှာဖွေခြင်း
	Linker က program ရဲ့ .dynamic section ကို ကြည့်တယ်
	DT_NEEDED entries တွေထဲမှာ ဘယ် libraries တွေ လိုအပ်သလဲ ဆိုတာ ရှာတယ်

#### Libraries Load လုပ်ခြင်း
	inker က LD_LIBRARY_PATH, /etc/ld.so.cache, နဲ့ default library paths တွေကနေ 
	libraries တွေကို ရှာတယ်
	libc.so.6 ကို တွေ့တဲ့အခါ → libc-2.23.so ကို memory ထဲ load လုပ်တယ်
	လိုအပ်တဲ့ library အားလုံးကို load လုပ်ပြီး memory mapping လုပ်တယ်

#### Relocation လုပ်ခြင်း

	 Symbol resolution လုပ်တယ်
	 Function addresses တွေကို fix လုပ်တယ်
	 build-in function call လုပ်တိုင်းလုပ်တယ်
	GOT (Global Offset Table) နဲ့ PLT (Procedure Linkage Table) တွေကို update လုပ်တယ်
	 Example:
	 printf function ရဲ့ actual address ကို libc ထဲမှာ ရှာတယ်
	 Program ရဲ့ PLT entry ထဲမှာ ထည့်တယ်

#### Initialization လုပ်ခြင်း

	 Libraries တွေရဲ့ initialization functions တွေကို run တယ်
	_init() functions တွေ ခေါ်တယ်
	__libc_start_main() ကို ခေါ်တယ် (libc ထဲမှာ)

#### Program Execution

	ခုဆိုရင် program က အလုပ်လုပ်ဖို့ အဆင်သင့်ဖြစ်ပြီ
	 Function call တစ်ခုခု (ဥပမာ printf()) လုပ်တိုင်း:
	 program → PLT → GOT → libc ထဲက actual function

## Step 10: Program အဆုံးသတ်ခြင်း

	main() return ပြန်တဲ့အခါ
	exit() function ကို ခေါ်တယ်
	Cleanup functions (_fini()) တွေ run တယ်
	Memory တွေ free လုပ်တယ်
	System ကို exit code ပြန်ပို့တယ်
